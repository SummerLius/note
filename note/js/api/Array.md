#### Array.from()

该方法从一个类似数组或可迭代的对象中创建一个新的数组实例。

`Array.from(arrayLike[, mapFn[, thisArg]])`

- arrayLike：
- mapFn：
- thisArg：
- @return：一个新Array实例

