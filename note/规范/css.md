# CSS规范

## 规范要求

1. 规范目标
    - 可重用
    - 可读性
    - 可维护性
2. 规范操作方向
    - css起名
    - css选择器级联结构
    - css属性分类规划（例如宽高属性独立出来单独成立一个格栅系统）
    - ...
3. 工具
    - sass
    - ...
