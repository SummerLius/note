2. **公共语言运行时CLR**
    - .NET Framework 的中心是其运行执行环境，叫做公共语言运行时（Common Language Runtime，CLR）或 .NET运行时（.NET Runtime）。在CLR控制下运行的代码通常被称为托管代码（managed code）。
    - 在CLR执行之前，源码必须经过编译。.NET下的编译有两个步骤：
        1. 源码编译成中间语言（Intermediate Language，IL）
        2. 中间语言被CLR编译成特定平台代码
