## String

- 字符串类型是零个或多个16位无符号整数值的有序序列，其最大长度为 2^53 -1。
- 在运行的 ECMAScript 程序中，字符串类型常被用于表示文本数据，此时字符串中的每个元素都被视为一个 UTF-16 编码单元。
- > 关于如何es程序如何将字符串中每个元素识别为utf16编码单元，暂时不深究。
