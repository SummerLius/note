# 7zr

## 概述

- 语法
    - `7zr [adeltux] [-] [SWITCH] <ARCHIVE_NAME> <ARGUMENTS>...`
- 描述
    - 7-zip是压缩率很高的文件压缩工具。该工具支持7z，lzma2，xz，zip，zip64，cab，rar，arj，gzip，bzip2，tar，cpio，rpm，iso，以及大多文件系统镜像和deb格式。
- 功能命令
    - `a`：添加
    - `d`：删除
    - `e`：解压
    - `l`：列出显示
    - `t`：测试
    - `u`：更新
    - `x`：以全路径解压
