# 数学

## 概念

数学是利用符号语言研究[数量](#)、[结构](#)、[变化](#)以及[空间](#)等概念的一门学科，从某种角度看属于形式学科的一种。数学通过抽象化和逻辑推理的使用，由计数、计算、量度和对物体形状及运动的观察而产生。
